7. Claves para que sea EL editor definitivo

Solo bullets prácticos:

Drag & drop todo (recetas, bloques, días enteros).

Recetas inteligentes: cada receta con macros, tags, restricciones, coste aproximado.

Objetivos claros: siempre visibles kcal + macros objetivo día/semana.

IA útil, no postureo:

Generar menú completo según objetivo y gustos.

Sustituir recetas manteniendo macros.

Detectar violaciones de restricciones.

Consistencia visual con el editor de entrenos:

Misma paleta, gradientes suaves en headers, cards blancas, sombras ligeras.

Mismo layout: izquierda biblioteca, centro vista principal, derecha resumen.

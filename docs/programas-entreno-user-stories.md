# Programas de Entreno · User Stories

## Topbar
- Como **coach**, quiero abrir la ficha del cliente desde un botón de información para consultar rápidamente sus datos, objetivos y restricciones.
- Como **coach**, quiero lanzar un asistente "Fit Coach" que se abra en un pop-up con recomendaciones IA para optimizar el plan del día.
- Como **coach**, quiero disponer de un botón de "Sustituciones de ejercicios" que abra un pop-up para reemplazar ejercicios en bloque sin navegar por la biblioteca manualmente.
- Como **coach**, quiero usar la acción "Batch Training" desde la topbar para aplicar ediciones masivas automáticas al plan semanal.

## Panel lateral izquierdo (Biblioteca & bloques)
- Como **coach**, quiero anclar plantillas favoritas en la parte superior de la biblioteca para reutilizarlas con un solo clic.
- Como **coach**, quiero filtrar la biblioteca por equipamiento disponible en la sede del cliente para evitar ejercicios inviables.
- Como **coach**, quiero ver etiquetas de dificultad 

## Área central (Canvas)
- Como **coach**, quiero reorganizar bloques dentro de la vista semanal mediante drag & drop para ajustar el orden de las sesiones.
- Como **coach**, quiero ajustar objetivos de duración y calorías por día desde la vista Excel para recalcular totales al instante.
- Como **coach**, quiero recibir indicadores visuales cuando un día excede los objetivos semanales para tomar decisiones informadas.

## Panel lateral derecho (Resumen)
- Como **coach**, quiero ver sugerencias contextuales distintas según la vista activa (semanal/diaria/Excel) para obtener insights adaptados.
- Como **coach**, quiero registrar pequeñas notas de seguimiento en el panel derecho para recordar ajustes planteados con el cliente.
- Como **coach**, quiero establecer objetivos semanales manualmente y que el panel recalculé el progreso según esos valores.

## Interacciones generales
- Como **coach**, quiero que los paneles recuerden su estado (colapsado/expandido) cuando vuelva a la página para continuar donde lo dejé.
- Como **coach**, quiero que los atajos de teclado permitan cambiar de vista y colapsar paneles para trabajar más rápido sin usar el ratón.
